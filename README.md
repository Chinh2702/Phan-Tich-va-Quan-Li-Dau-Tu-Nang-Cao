# 📊 Advanced Investment Analysis and Portfolio Management

## 📌 Overview
This repository documents my learning and practical applications in advanced investment analysis and portfolio management. It integrates financial theory (CFA-oriented) with quantitative and data-driven approaches, including portfolio optimization, asset pricing, and financial machine learning.

---

## 🎯 Objectives
- Develop a solid understanding of risk-return dynamics and downside risk measures  
- Apply modern portfolio theory and asset pricing models (CAPM, APT, Fama-French)  
- Conduct equity and fixed income valuation using fundamental techniques  
- Analyze macroeconomic and industry factors affecting investment decisions  
- Build and backtest systematic trading strategies using machine learning  
- Evaluate portfolio performance under industry-standard metrics and ethical frameworks  

---

## 🧠 Key Topics

### 1. Risk and Return
- Compounding, annualization, inflation-adjusted returns  
- Volatility, Value at Risk (VaR), Expected Shortfall, tail risk  

### 2. Portfolio Theory
- Markowitz Efficient Frontier and diversification  
- Capital allocation and risk decomposition  
- Asset pricing models: CAPM, APT, Fama-French  

### 3. Security Analysis & Valuation
- Equity valuation: DDM, P/E, Free Cash Flow (FCF)  
- Financial statement analysis: DuPont framework  
- Fixed income: bond pricing, yield curve, duration & convexity  

### 4. Investment Strategies
- Market efficiency vs. behavioral finance  
- Alternative assets: hedge funds, private equity, real estate  
- Technical analysis and signal-based strategies  

### 5. Financial Machine Learning
- Data structuring: Tick, Volume, Dollar Bars  
- Labeling techniques: Triple-Barrier Method  
- Model validation: Purged K-Fold Cross-Validation  
- Portfolio optimization: Hierarchical Risk Parity (HRP)  

### 6. Portfolio Management & Ethics
- Performance evaluation: Sharpe Ratio, Jensen’s Alpha, Treynor Ratio  
- Investment process aligned with CFA framework  
- Ethical standards and GIPS compliance
